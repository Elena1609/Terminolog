# Терминолог

🔗 **Живое приложение:** [terminolog.replit.app](https://terminolog.replit.app/)

Учебный проект — AI-агент на базе библиотеки [smolagents](https://github.com/huggingface/smolagents), который предлагает определения терминов в стиле ГОСТов.

## Описание

Агент получает название термина, выполняет поиск в интернете через DuckDuckGo, при необходимости читает найденные страницы и формулирует определение по правилам ГОСТ (структура: «Термин — это ...»).

Проект доступен в двух режимах:
- **Веб-интерфейс** (`server.py`) — Flask-приложение с браузерным UI
- **Консоль** (`main.py`) — интерактивный цикл ввода в терминале

## Возможности

- Одиночный запрос — введи термин, получи определение
- Пакетный режим — загрузи `.txt`-файл со списком терминов, агент обработает их по очереди с прогресс-баром
- Сохранение результатов в `.txt`-файл (одно определение или все сразу)
- История запросов за текущую сессию

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git
cd ВАШ_РЕПОЗИТОРИЙ

# Установить зависимости
pip install smolagents==1.12.0 "smolagents[openai]" flask flask-cors python-dotenv ddgs
```

## Настройка

Создайте файл `.env` в корне проекта:

```env
OPENAI_API_KEY=ваш_ключ_openrouter
OPENAI_BASE_URL=https://openrouter.ai/api/v1
```

> Ключ API получить на [openrouter.ai](https://openrouter.ai). Файл `.env` добавлен в `.gitignore` — в репозиторий не попадёт.

## Запуск

**Веб-интерфейс:**
```bash
python server.py
```
Откройте [http://localhost:5000](http://localhost:5000)

**Консольный режим:**
```bash
python main.py
```
Введите термин и нажмите Enter. Для выхода введите `выход`.
